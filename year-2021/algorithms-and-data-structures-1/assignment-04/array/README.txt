Zeby uruchomić program, proszę wpisać:
make
./ex4Array

Operacje dominujące:

    push:
        stackArray[elementsIn] = element;
        O(1);

    pop:
        for;
        O(n);
    
    peek:
        return;
        O(1);
    
    isEmpty:
        if;
        O(1);
    
    size:
        return;
        O(1);

    isFul:
        return;
        O(1);
